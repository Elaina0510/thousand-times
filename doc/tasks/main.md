# main.py — 主程序模块

> 最小可执行任务列表，用于 Vibe Coding

---

## 任务 1：基础框架

- [ ] 创建 `src/main.py` 文件
- [ ] 配置日志系统（logging，级别 INFO/WARNING/ERROR）
- [ ] 实现 `main()` 函数入口

## 任务 2：流水线编排

- [ ] 调用 `load_config()` 加载配置
- [ ] 调用 `get_stock_pool()` 获取股票池
- [ ] 调用 `get_etf_pool()` 获取ETF池
- [ ] 调用 `fetch_news()` + `analyze_policy_impact()` 抓取并分析新闻（全局一次）

## 任务 3：个股分析循环

- [ ] 遍历股票池，逐只调用 `analyze_single_stock()`
- [ ] 实现 `analyze_single_stock()` 函数（技术指标→基本面→政策→行业→综合评分）
- [ ] 筛选：总分 ≥ 70 或 < 30 的进入推送列表
- [ ] 异常隔离：单只失败记录警告，继续分析其他

## 任务 4：ETF分析循环

- [ ] 遍历ETF池，逐只调用 `analyze_single_etf()`
- [ ] 实现 `analyze_single_etf()` 函数（技术指标→政策→资金流向→综合评分）
- [ ] 筛选：总分 ≥ 70 或 < 30 的进入推送列表
- [ ] 异常隔离：单只失败记录警告，继续分析其他

## 任务 5：图表与报告

- [ ] 对推送列表中的每只股票/ETF生成图表
- [ ] 调用 `generate_report()` 生成报告文本

## 任务 6：推送

- [ ] 从环境变量读取 `PUSHPLUS_TOKEN`
- [ ] 调用 `push_to_wechat()` 推送报告
- [ ] 推送失败时记录错误日志

## 任务 7：集成测试

- [ ] 测试完整流程各模块按序调用
- [ ] 测试单只股票失败时跳过继续
- [ ] 测试全部失败时生成空报告
- [ ] 测试推送失败时记录错误
